# Code of Conduct

## Our standard

We expect participation in this repository to stay respectful, practical, and focused on improving the product experience for everyone.

Examples of expected behavior:

- Be clear and factual
- Assume good intent
- Report problems with enough detail to reproduce them
- Keep discussion focused on the published product

Examples of unacceptable behavior:

- Harassment or personal attacks
- Hate speech or discriminatory language
- Spam, trolling, or repeated bad-faith behavior
- Posting private data, secrets, or exploit details in public threads

## Scope

This code of conduct applies to:

- Issues
- Pull requests
- Release discussions
- Other public interactions in this repository

## Enforcement

The repository owner may remove content or restrict participation that violates these expectations.

## Security note

If your report involves a sensitive vulnerability, use the process in [SECURITY.md](SECURITY.md) instead of posting it publicly.
