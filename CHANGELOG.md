# Changelog

All notable changes to the public distribution repo and published desktop releases are documented here.

## v0.1.1

Release date: March 10, 2026

### Added

- First public desktop preview for the OpenClaw + Telegram setup path
- Quick Start flow for environment check, OpenClaw install/verify, and Telegram bridge setup
- Manual Mode for the currently supported OpenClaw actions
- Local receipts for setup and bridge actions
- Public distribution repo with screenshots, release assets, checksums, and support docs

### Current limits

- macOS Apple Silicon (`arm64`) only
- Pre-release build
- Not notarized yet
- Discord and Lark / Feishu are not included in this release
