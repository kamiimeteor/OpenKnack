# Contributing

This repository is a **public distribution repo** for AI Gene Capsule releases.

## What this repo is for

- Publishing desktop release assets
- Hosting release notes and download instructions
- Collecting bug reports, install problems, and feature requests
- Explaining product scope and support boundaries

## What this repo is not for

- Source development
- Source-code pull requests
- Internal implementation discussion
- Private roadmap or unreleased implementation details

## Contributions we welcome here

- Bug reports against published desktop releases
- Install and launch issue reports
- Product feedback and feature requests
- Documentation corrections for the public distribution materials

## Pull requests

Small documentation-only pull requests may be reviewed.

Source-code PRs are out of scope for this repository because this is not the development repository.

## Before opening an issue

- Confirm the app version you are using
- Check the [README](README.md) troubleshooting section
- Check the current [release notes](https://github.com/kamiimeteor/AI-Gene-Capsule/releases)
- Avoid posting secrets such as Telegram bot tokens
